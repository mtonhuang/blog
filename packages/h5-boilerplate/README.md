

# 基于webpack,less开发的h5重构项目脚手架

github的weui是基于gulp进行搭建的，因业务需要我重新用webpack搭建了一个。

这个模板主要针对重构，因此没有使用webpack多个入口进行配置，采用了显示隐藏来控制，简单方便。

点击进入想要体验的页面，需要退回时刷新即可,有任何问题欢迎提issues骚扰 —— mtonhuang
