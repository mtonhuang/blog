

# github的weui是基于gulp进行搭建的，因业务需要我重新用webpack搭建了一个

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

这个模板主要针对重构，因此没有使用webpack多个入口进行配置，采用了显示隐藏来控制。
